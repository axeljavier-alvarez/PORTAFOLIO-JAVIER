Qué hace el proyecto

Este proyecto es un CRUD de empleados desarrollado con Laravel y XAMPP/MySQL. Permite agregar, editar, eliminar y visualizar los datos de los empleados en la base de datos. La información se almacena en MySQL y se puede gestionar de manera sencilla desde el navegador.

Estructura del proyecto en Laravel

El proyecto está construido sobre el framework Laravel, donde se crearon:

  • Controladores y modelos para manejar la lógica de los empleados.
  • Migraciones para definir los campos de la base de datos.
  • Vistas Blade conectadas con rutas y controladores.
  • Autenticación de usuarios con login (correo y contraseña).
  • UI y Bootstrap para mejorar la presentación visual.
Todo esto permite un flujo de trabajo completo y organizado dentro del proyecto.
